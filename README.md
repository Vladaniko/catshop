# Catshop (training web app)

Учебный сайт-магазин котиков на Python (Flask) + SQLite: регистрация/логин, комментарии, личные сообщения, поиск пользователей и друзья.

## Важно

Этот проект предназначен **только для локальной тренировки** и учебных стендов. Не выкатывай его в интернет.

## Запуск

```bash
cd "/home/karmi/Documents/debian/Catshop"
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

Открой в браузере `http://127.0.0.1:5000`.

## Функции

- Регистрация/логин
- Главная: каталог котиков (Catshop)
- Комментарии к котикам
- Профили пользователей + поиск
- Добавление/удаление друзей
- Личные сообщения
